Youtube Comments App
I recently completed a really cool project using the YouTube API and machine learning algorithms. It's a data analysis dashboard that anyone can use, built with the Streamlit framework in Python. The project is mainly used for my own game design devlogs, and it solves the problem of quickly going through comments and seeing how they're received.

One of the coolest things about this project is that it uses machine learning algorithms for sentiment analysis. In simple terms, this means that it can tell whether a comment is positive, negative, or neutral, and this helps me get a better understanding of how my audience is responding to my videos. By using this kind of technology, I can quickly identify patterns and trends in the comments, and this makes it easier for me to create content that resonates with my viewers.

Overall, I'm really proud of this project because it shows that I can create apps that anyone can use, and it allows me to work more efficiently by automating the process of analyzing comments. Plus, it's just really cool to see how technology can be used in this way to solve real-world problems.
